# 🇨🇦 Canadian Grand Prix 2025 — Race Analytics

Circuit Gilles Villeneuve, Montreal | Round 9 of 24

Deep-dive F1 race analysis powered by FastF1.
Covers lap time evolution, tyre strategies, speed telemetry,
sector breakdowns, pit stops, weather impact and position battles
for all 20 drivers — with Ferrari's HAM vs LEC head-to-head focus.

## Built With
- FastF1 — official F1 telemetry data
- Matplotlib — dark theme F1 visualizations  
- Pandas & NumPy — data processing

## Run It
pip install -r requirements.txt
jupyter notebook canada_gp_2025_analysis.ipynb
